[Início](python-curso)
```python
nome = input("Digite seu nome: ")

# Perguntar ao usuário sua idade e armazenar
idade = int(input("Digite sua idade: "))

# Imprimir o nome do usuário em maiúsculas
print(nome.upper())

# Imprimir qual o ano de nascimento do usuário
print(f'Você nasceu em {2022-idade}')
```
